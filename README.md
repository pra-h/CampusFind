# CampusFind

**CampusFind** is a full-stack campus-based Lost & Found management platform designed to streamline the reporting, claiming, and verification of misplaced items within a college campus.

## Key Features
- Role-based access for Students and Admins  
- OTP-based authentication for secure student registration  
- Real-time lost item reporting and claim verification  
- Admin-controlled claim approval workflow  
- Automated email notifications for status updates  

## Technology Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: OTP, JWT  
- Email Service: Automated email notifications  

## Project Scope
This repository contains the **frontend implementation and project documentation only**.  
Backend logic, database configurations, and credentials are excluded for security and academic integrity.
